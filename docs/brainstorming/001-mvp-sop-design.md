# 视频二创 Agent - MVP 设计文档

**创建日期:** 2025-03-19  
**版本:** v0.1 MVP  
**状态:** 已确认

---

## 一、产品定位

### 目标用户
- **MVP 阶段:** 管理员单人使用（本地部署）
- **后续阶段:** 多用户 SaaS 服务（付费订阅）

### 核心价值
用户上传本地视频 → AI 自动完成语音转录、内容分析、二创改写、审核评估 → 输出可直接用于短视频平台发布的完整稿子

## 二、完整 SOP 流程

```
┌──────────────────────────────────────────────────────────────────┐
│                        视频上传                                   │
└──────────────────────────────────────────────────────────────────┘
                                ↓
┌──────────────────────────────────────────────────────────────────┐
│                    语音提取 + 转录 (Whisper)                      │
│              - 提取音频轨道                                       │
│              - Whisper medium 模型转录                            │
│              - 输出带时间戳的文字稿                               │
└──────────────────────────────────────────────────────────────────┘
                                ↓
┌──────────────────────────────────────────────────────────────────┐
│                      内容分析 (Qwen)                              │
│              - 提取核心观点和重点内容                             │
│              - 联网搜索相关热门话题                               │
│              - 生成标签/话题建议                                  │
└──────────────────────────────────────────────────────────────────┘
                                ↓
┌──────────────────────────────────────────────────────────────────┐
│              二创改写 - 生成 3 个版本 (DeepSeek)                     │
│              - 对原内容质疑                                      │
│              - 观点分析                                         │
│              - 观点补充                                         │
│              - 伪原创（换语法/口吻/比喻）                         │
│              - 输出 3 种不同表达方式的版本                          │
└──────────────────────────────────────────────────────────────────┘
                                ↓
┌──────────────────────────────────────────────────────────────────┐
│                  平台适配优化 (Qwen)                              │
│              - 根据目标平台调整开头 (抖音/小红书/B 站)              │
│              - 调整节奏和时长                                     │
│              - 联网搜索最新平台规则                               │
└──────────────────────────────────────────────────────────────────┘
                                ↓
┌──────────────────────────────────────────────────────────────────┐
│                      AI 审核 (Qwen)                               │
│              - 原创度评分 (0-100)                                 │
│              - 合规性评估                                         │
│              - 吸引力评分 (低/中/高)                              │
│              - 敏感词扫描                                         │
│              - 修改建议                                           │
└──────────────────────────────────────────────────────────────────┘
                                ↓
┌──────────────────────────────────────────────────────────────────┐
│                      人工审核                                     │
│              - 管理员审查 3 个版本                                  │
│              - 选择最佳版本或提出修改意见                         │
└──────────────────────────────────────────────────────────────────┘
                                ↓
┌──────────────────────────────────────────────────────────────────┐
│                    逐字稿生成                                     │
│              - Markdown 格式输出                                   │
│              - 包含语气标注                                       │
│              - 包含分镜建议                                       │
│              - 包含重点表现方式                                   │
└──────────────────────────────────────────────────────────────────┘
                                ↓
┌──────────────────────────────────────────────────────────────────┐
│              标题/封面文案生成                                    │
│              - 生成 10 个标题变体                                   │
│              - 包含封面文案建议                                   │
│              - A/B 测试版本                                       │
└──────────────────────────────────────────────────────────────────┘
                                ↓
┌──────────────────────────────────────────────────────────────────┐
│              标签建议 + 发布时间建议                              │
│              - 推荐 5-10 个热门标签                                │
│              - 根据平台推荐最佳发布时间                           │
└──────────────────────────────────────────────────────────────────┘
                                ↓
┌──────────────────────────────────────────────────────────────────┐
│                        输出                                       │
│              - Markdown 稿子文件                                   │
│              - 标题库 JSON                                        │
│              - 运营建议汇总                                       │
└──────────────────────────────────────────────────────────────────┘
```

## 三、技术架构

### 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| **前端** | HTML + TailwindCSS + Alpine.js | 轻量级，无需构建 |
| **后端** | Python FastAPI | API 服务 + 静态文件服务 |
| **数据库** | SQLite | 本地文件数据库 |
| **语音转录** | Whisper (medium) | 本地运行 |
| **AI 模型** | Qwen3.5 (plus/flash) | 内容分析/审核/搜索 |
| **AI 模型** | DeepSeek V3.2 | 二创改写 |

## 四、用户系统

### MVP 阶段
- **单用户:** 管理员账号（写死）

- **权限:** 全部功能

### 后续扩展（v0.2+）
- **多用户:** 数据库用户表
- **角色:** admin / user
- **权限:** 基于角色的访问控制

## 五、模型配置

### 模型分工

| 环节 | 模型 | 理由 |
|------|------|------|
| 内容分析 | Qwen3.5-flash | 速度快、省钱 |
| 二创改写 | DeepSeek V3.2 | 逻辑推理强、核心质量 |
| 平台适配 | Qwen3.5-flash | 快 |
| AI 审核 | Qwen3.5-plus | 准确性优先 |
| 标题生成 | Qwen3.5-flash | 快 |

### 配置方式

```bash
# .env
QWEN_API_KEY=sk-xxx
QWEN_BASE_URL=https://dashscope.aliyuncs.com/api/v1
DEEPSEEK_API_KEY=sk-xxx
DEEPSEEK_BASE_URL=https://api.deepseek.com/v1

# 默认模型
DEFAULT_ANALYSIS_MODEL=qwen-flash
DEFAULT_REWRITE_MODEL=deepseek-v3.2
DEFAULT_REVIEW_MODEL=qwen-plus
```

## 六、数据库设计

### users 表
```sql
CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    username TEXT UNIQUE NOT NULL,
    password_hash TEXT NOT NULL,
    role TEXT DEFAULT 'user',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

### videos 表
```sql
CREATE TABLE videos (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER NOT NULL,
    file_path TEXT NOT NULL,
    original_filename TEXT NOT NULL,
    duration INTEGER,
    status TEXT DEFAULT 'pending',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

### scripts 表
```sql
CREATE TABLE scripts (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    video_id INTEGER NOT NULL,
    version INTEGER DEFAULT 1,
    content TEXT NOT NULL,
    titles TEXT,
    tags TEXT,
    suggestions TEXT,
    status TEXT DEFAULT 'draft',
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (video_id) REFERENCES videos(id)
);
```

## 十、风险与假设

### 技术风险
- Whisper 本地运行需要一定计算资源
- 大视频文件转录时间较长

### 假设
- 用户有 Qwen 和 DeepSeek API 账号
- 用户本地有足够存储空间
- 二创内容不涉及严重版权问题（MVP 阶段）
